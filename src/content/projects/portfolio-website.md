This is my personal portfolio website showcasing my projects and blog posts. Built with modern web technologies for optimal performance and user experience.

## Features

- **Responsive Design**: Works perfectly on all devices
- **Dynamic Content**: Posts and projects loaded from markdown files
- **Fast Loading**: Optimized with Vite and modern bundling
- **Accessible**: Built with accessibility best practices

## Technologies Used

- React 19 with TypeScript
- Tailwind CSS for styling
- Vite for build tooling
- React Router for navigation
- React Markdown for content rendering
